## Description
Creating an App to book workplaces and meetingrooms. 

------------

### Built With
- NodeJs
- Express
- MongoDB
- Mongoose
- Axios
- Cors
- Bcrypt
- pg
- dotenv
- jsonwebtoken

------------

### Installation
npm i 

------------

### Start the App
nodemon

------------

### Features
- User Collection
- Place Collection
- Reservation Collection
- Protected Routes
